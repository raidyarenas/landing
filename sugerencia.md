## 5. Experiencia Laboral

### Tips para cada posición:

1. **Usa verbos de acción:** Diseñé, Desarrollé, Lideré, Implementé, Optimicé
2. **Incluye métricas cuando sea posible:**
   - "Reduje el tiempo de carga en un 40%"
   - "Aplicación con +1,000 descargas en Google Play"
   - "Sistema usado por +50 restaurantes"
3. **Menciona el stack en cada rol**

### Ejemplo de descripción:

```
Fullstack Developer @ Reservándonos
[Fecha inicio] - Presente

- Diseñé y desarrollé 2 aplicaciones móviles híbridas (Ionic/Capacitor) publicadas en Google Play Store
- Arquitecté el backend con Node.js y Express.js, implementando APIs RESTful
- Construí el frontend administrativo con Angular y TailwindCSS
- Desplegué y mantuve la infraestructura en AWS (EC2, S3, RDS)
- Sistema de reservas en tiempo real para restaurantes con WebSockets

Stack: Node.js, Angular, Ionic, Capacitor, TypeScript, MySQL, AWS
```

## 9. Configuración de Perfil

### Open to Work:
1. Ve a tu perfil → "Open to" → "Finding a new job"
2. Configura:
   - **Job titles:** Fullstack Developer, JavaScript Developer, Node.js Developer, Angular Developer
   - **Location:** Remote, o países específicos
   - **Start date:** Immediately / In 2 weeks
   - **Job types:** Full-time, Contract

### Visibilidad:
- Elige "All LinkedIn members" para máxima visibilidad
- O "Recruiters only" si prefieres discreción

# Raidy Alejandro - Landing Page

Portfolio personal desplegado en Cloudflare Pages.

## Stack

- HTML5
- TailwindCSS (CDN)
- JavaScript vanilla

## Deploy

Este sitio está configurado para Cloudflare Pages. Solo necesita archivos estáticos, sin build.

## Archivos

- `index.html` - Página principal
- `404.html` - Página de error
- `_headers` - Configuración de headers para Cloudflare
- `sugerencia.md` - Notas para optimización de LinkedIn
